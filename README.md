# ROMA-SPQR / ROME-YES

Vite React deployment for Railway.

## Railway
Build Command:
```bash
npm install && npm run build
```
Start Command:
```bash
npm run start
```

## Notes
- Do not upload `node_modules`, `dist`, `.env`, `.env.local`, `.DS_Store` or `package-lock.json`.
- Shared game data uses the Supabase REST setup via `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
