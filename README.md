# ROME-YES — Reputation & Slander Update

This update adds a player-facing Reputation system:

- Personal → Reputation section
- Senators can spend private wealth to increase their reputation
- Senators can spend private gold to slander rivals
- Automatic roll for slander outcomes
- Success/partial success damages target reputation and adds a public scandal marker
- Backfire exposes the slanderer and creates grounds for Praetor Urbanus court action
- Slandered senators can spend gold on counter-campaigns to reduce/remove scandal impact
- Game Master can see and edit reputation scores, scandal marks, prices, success chances and effects
- Senator profiles now show reputation and active scandals

Do not upload node_modules, dist, .env, .env.local or package-lock.json.
