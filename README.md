# ROME-YES / SPQR Webapp

Vite + React app for the Discord Roman Republic campaign.

## Latest update

- Senator-to-senator transfers of private gold and food.
- Senator-to-senator property transfers.
- Household upkeep in gold and food for each senator.
- Office salaries by magistracy, paid when the GM advances the season.
- Biography/background field for each senator profile.
- Political parties: create a party, invite members, join/leave, and display party badges in the Senate.
- GM Private Wealth panel can adjust household upkeep and regional business slots.
- Backup/restore includes parties and all wealth/property data.

## Railway variables

Keep these in Railway:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Do not upload `node_modules`, `dist`, `.env`, `.env.local`, or `package-lock.json`.
